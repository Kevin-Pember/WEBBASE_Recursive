Memory Functions

MRC works to both recall and deletes a recorded number. To first recall a number press the MRC button then the number will appear in the entry field. To use the MRC button to delete the recorded value, press it when the only value in the entry field

M or memory add button adds whatever is in the entry field to the recorded value, this includes any operators so if there is a multiplication sign in front of the number it will then it multiply the recorded value by that number

Rad and Deg Modes

Whatever the button’s currently displaying is what mode it's in, press it to change it. Whenever recursive starts it defaults to degrees.

Parenthesis button behavior

When you press this button depending on if there is or isn’t already a front parenthesis it will type the corresponding sign needed to start or end a parenthesis statement.

If there isn’t a front parenthesis ‘(’ or all parenthesis before caret position already form a complete statement or both a front and back parenthesis the button will type a front parenthesis. It will also type a front parenthesis if the character before the caret is also a front parenthesis.

If the conditions do not meet any of the conditions stated above the parenthesis button with type a back parenthesis ‘)’.

Power of Button

The Power of Button changes all functions into their power of equivalent forms which will be processed at solve time. To get of power of mode press the arrow right key.

Decimal to Fration function

Otherwise noted as d→f, this function converts a decimal to a fraction

Delete History Button

This is the button with a trash can and an arrow pointing down

Settings

Colors

The colors section of colors is for choosing the individual colors of certain parts of recursive. This gives you more power to customize recursive.

The themes section of recursive has the preset themes of dark and light with more on the way.

Preferences

Output Length is the length of numbers that recursive returns. If the number recursive solves is longer than this length it will change the number into the scientific notation of that number.

Default domain and range are the default values that a graph custom function goes to by default. These values are only the default and can be changed in the custom graph itself
